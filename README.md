## Running the both backend and frontend
1. Clone the repository
2. On your terminal, go to the cloned repository
3. Install dependencies:

    ```
    npm i
    ```

4. Copy env:

    ```
    npm run setup
    ```
5. Run the application:

    ```
    npm start
    ```
6. Navigate to `http://localhost:3000/`
